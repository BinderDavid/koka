fun decimal( i : intstd/core/int: V, frac : ?intstd/core/int: V, prec : ?intstd/core/int: V ) : decimalstd/num/decimal/decimal: V

Create a decimal number with precision prec (=-1) from a whole part i and fractional part frac. The result is i + frac×10-prec. i.e. prec determines how many digits there are behind the dot. decimal(1,123,4).show == "1.0123"
decimal(1,123).show == "1.123"
decimal(123).show == "123"
If prec is negative or smaller than the number of digits in frac, the result will take the width of frac in digits as its precision.
decimal(1,123,2).show == "1.123".

Create a decimal from a doublestd/core/double: V with a specified maximal precision (=16 which is the maximal precision of a doublestd/core/double: V). Note: creating a decimalstd/num/decimal/decimal: V from a doublestd/core/double: V may loose precision and give surprising results as many decimal fractions cannot be represented precisely by a doublestd/core/double: V. Use decimal-expstd/num/decimal/decimal-exp: (i : int, exp : ?int) -> decimal or parse-decimalstd/num/decimal/parse-decimal: (s : string) -> maybe<decimal> when exact representations are required. For example: 

> decimal(1.1)
1.1000000000000001
> decimal(1.2)
1.2
> decimal(1.1)
1.1000000000000001
> decimal(1.2)

fun div( x : decimalstd/num/decimal/decimal: V, y : decimalstd/num/decimal/decimal: V, min-prec : ?intstd/core/int: V ) : decimalstd/num/decimal/decimal: V

Divide two decimals with a given extra precision min-prec (=15). The min-prec is the number of extra digits used to calculate inexact divisions.

Note: the division uses up to min-prec precision using Floor rounding for the last digit if the result is inexact. To round differently, you can for example divide with larger precision and use round-to-prec. 

> div( decimal(2), decimal(3), 0 )
0
> div( decimal(2), decimal(3), 1 )
0.6
> div( decimal(2), decimal(3) )  // default precision is 15
0.6666666666666666
> div( decimal(2), decimal(3) ).round-to-prec(6)
0.666667
> divstd/num/decimal/div: (x : decimal, y : decimal, min-prec : ?int) -> decimal( decimal(2), decimal(3), 0 )

> divstd/num/decimal/div: (x : decimal, y : decimal, min-prec : ?int) -> decimal( decimal(2), decimal(3), 1 )

> divstd/num/decimal/div: (x : decimal, y : decimal, min-prec : ?int) -> decimal( decimal(2), decimal(3) )  // default precision is 15
